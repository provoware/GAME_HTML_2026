## Entwicklungsumgebung

Für die weitere Implementierung wird Node.js 24 LTS verwendet. Die Version ist in `.nvmrc`
festgelegt, damit lokale Entwicklung und spätere CI-Läufe dieselbe Node-Hauptversion nutzen.

Der aktuelle Container kann davon abweichen; vor npm-, Vite- oder Testarbeiten ist deshalb die
aktive Node-Version gegen `.nvmrc` zu prüfen.
